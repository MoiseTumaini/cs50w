# Django

- Used to make websites more dynamic as thing change in your web application

- The protocol that used for request/response is called http

HTTP Status Code

<ul>
    <li>200 --- OK </li>
    <li>301 --- Moved Permanently </li>
    <li>403 --- Forbidden </li>
    <li>404 --- Not Found </li>
    <li>500 --- Internal Server Error </li>
</ul>
